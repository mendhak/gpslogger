轻量、省电的 GPS 记录器。 此程序用于定期记录您的 GPS 座标到 SD 卡上的文件中。  此程序可在内存后台运作；无论您是散步、远行、飞行还是摄影会甚至是出门买牛奶，本程序都可持续记录gps轨迹。   当您回到电脑前，即可使用这些文件以标记照片坐标，或是上传至旅行网站，也可于谷歌地球中查看。 

***的特性：

* 指定记录间隔
* 指定路程间隔
* 移动手机基站或 GPS 卫星定位
* 支持GPX、KML 文件
* 显示通知
* 使用手机系统时间或卫星授时
* 显示英制单位
* 开机后自启程序
* 定时通过电子邮件发送记录
* OpenStreetMap——上传 GPS 路径
* Dropbox——上传 GPX、KML、ZIP


有程序错误、功能请求或是疑问？请一并提交至 github.com/mendhak/gpslogger   

***备注：

GPSLogger 并非是“我的足迹”（OpenTracks）的替代品。  “我的足迹”适合短期使用（同时运行的功能很多），GPSLogger 可以长期运行。

本程序只在阁下要求电子邮件、OpenStreetMap 或 Dropbox“自动发送”功能时使用网络连接。

虽然本程序支持使用 0 秒间隔，但不建议使用，因为记录速度很快，可能会导致不稳定或卡机。 试着换成 1–3 秒吧。

如果您觉得本应用缺少某个该有的功能，您可在 GitHub 提交“功能请求”。

***权限解释：

存储：读写 SD 卡上的 GPSLogger 文件夹

网络：用于上传文件（Dropbox、Openstreetmap）、发送邮件，或是登录至 Dropbox/Openstreetmap

位置：用于获取您的 GPS 或基站位置

系统工具（开机自启）：在您选择开机自启时用到
