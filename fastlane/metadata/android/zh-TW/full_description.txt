輕量、省電的 GPS 記錄器。 此程式用於定期記錄您的 GPS 座標到 SD 卡上的檔案中。  此程式於背景後臺運作；無論您是散步、遠行、飛行還是攝影會甚至是出門買牛奶，本程式都可長期持續運行記錄。   只要您回到電腦前，即可使用這些檔案標記照片座標，或是上傳至旅行網站，也可於 Google 地球中查看。 

***特性：

* 指定記錄間隔
* 指定路程間隔
* 行動基地臺或 GPS 衛星皆可定位
* 支援匯出 GPX、KML 檔案
* 顯示通知
* 選擇使用手機或衛星時間
* 英制單位顯示
* 開機自啟、啟動時自動開始記錄
* 定時以電子郵件送出記錄
* OpenStreetMap——上載 GPS 路徑
* Dropbox——上載 GPX、KML、ZIP


有臭蟲、功能請求或是疑問？請一併遞交至 github.com/mendhak/gpslogger 的工單系統。  

***備註：

GPSLogger 並非一個「我的足跡」（MyTracks）替代品。  「我的足跡」適合短期使用（同時執行的功能很多），GPSLogger 細水長流。

本程式只在閣下要求電子郵件、OpenStreetMap 或 Dropbox「自動送出」功能時使用網路連接。

雖然本程式支援使用 0 秒間隔，但還是請勿使用：記錄速度很快，可能會導致不穩定或卡殼。 試著換成 1–3 秒吧。

如果您覺得本程式缺少某個該有的功能，您可於 GitHub 遞交「功能請求」。

***權限解釋：

儲存：讀寫 SD 卡上的 GPSLogger 資料夾

網路：用於上載檔案（Dropbox、Openstreetmap）、送出郵件，或是登錄至 Dropbox/Openstreetmap

位置：用於取得您的 GPS 或行動基地臺位置

系統工具（開機自啟）：在您選擇開機自啟時用到
